Pet Clinic Management Application Development Task

Overview
You are tasked with developing a simple application to manage clients of a Pet Clinic. The specifications of the system can be found in the provided "System Spec.svg" file. Additionally.
Project Requirements
●	Framework: Next.js or React Remix
●	Database: MongoDB
●	CSS: Tailwind CSS
Application Details
●	The application consists of a single page with a modal that has two variants: Add and Edit.
●	Implement the backend using Next.js API.
Guidelines
1.	Design and UI:
o	The design, colors, animations, icons, fonts, and overall aesthetics are up to you. Showcase your best implementation!
o	Make the app responsive to both mobile and desktop device.
2.	CSS and Styling:
o	Use Tailwind CSS exclusively..
o	Store CSS constants (colors, etc.) in tailwind.config.js.
o	Utilize generic spacing (e.g., p-4, p-8, p-16, p-32).
3.	Code Quality:
o	Write clean, reusable, and DRY (Don't Repeat Yourself) code.
o	Create small, modular components and functions.
o	Follow best practices for code readability and maintainability.
4.	Recommended Packages:
o	Mui
o	React Query
o	React Table
Feel free to use any other packages that enhance the development process and product quality.
5.	Project Management:
o	Document your progress, questions, design choices, and any challenges you face. Communication is key.
Deliverables
1.	Code Repository:
o	Create a GitHub repository for your project and provide the link.
2.	Deployment:
o	Deploy the application to a hosting platform of your choice (e.g., Heroku, Netlify, AWS).
o	Provide the deployment link.
Notes
●	Read the task description carefully; every instruction has a purpose.
●	We encourage you to ask questions and communicate any uncertainties you may have.
Submission
When you have completed the project, submit the GitHub repository link and the deployment link.
Good Luck!
We are excited to see your innovative solutions and high-quality code. If you have any questions, please do not hesitate to ask.
